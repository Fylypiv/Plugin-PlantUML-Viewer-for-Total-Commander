Інструкція користувача: PlantUML Viewer для Total Commander

Цей плагін дозволяє автоматично генерувати та переглядати UML-діаграми (PlantUML) безпосередньо у вікні переглядача (Lister) файлового менеджера Total Commander.

📌 Вимоги до системи

Файловий менеджер: Total Commander 64-bit.

Залежності: Встановлене середовище виконання Java (JRE) для локального рендерингу діаграм.

----------------------------------------------------------------------------------------------
🚀 Спосіб 1: Автоматичне встановлення (Рекомендовано)
Найшвидший спосіб інтегрувати плагін у систему:

1. Завантажте готовий інсталяційний архів PlantUml_Viewer.zip.

2. Відкрийте файловий менеджер Total Commander і перейдіть до директорії, куди ви завантажили архів.

3. Виділіть архів і натисніть клавішу Enter.

4. Total Commander розпізнає конфігураційний файл pluginst.inf і виведе вікно з пропозицією встановити плагін. Натисніть «Так».

5. За потреби підтвердьте шлях для розпакування (за замовчуванням програма сама створить папку в каталозі Plugins).

----------------------------------------------------------------------------------------------------------
🛠 Спосіб 2: Ручне встановлення
Якщо ви бажаєте встановити плагін вручну або змінити директорію:

1. Створіть нову папку для плагіна (наприклад, у піддиректорії plugins\wlx\PlantUmlViewer всередині папки встановлення Total Commander) і розпакуйте туди всі файли з архіву.

2. Відкрийте Total Commander. У верхньому меню перейдіть за шляхом: Конфігурація -> Налаштування (Configuration -> Options).

3. У лівій панелі оберіть розділ Плагіни (Plugins).

4. Знайдіть блок Плагіни внутрішнього переглядача (.WLX) і натисніть кнопку Налаштування (Configure).

5. У новому вікні натисніть кнопку Додати (Add) і через провідник виберіть скомпільований файл PlantUmlViewer.wlx64 (або PlantUML_Plugin_TC.wlx64) з папки, яку ви створили на першому кроці.

6. Натисніть ОК для підтвердження. Система автоматично зареєструє підтримувані розширення файлів.

----------------------------------------------------------------------------------------------------------
📖 Як користуватися плагіном

1. Знайдіть у Total Commander будь-який текстовий файл з кодом PlantUML. Плагін підтримує розширення: .puml, .plantuml, .uml, .wsd, .ws, .iuml.

2. Виділіть файл і натисніть клавішу F3 (Перегляд).

3. У дочірньому вікні миттєво з'явиться згенерована графіка діаграми.

Інструменти та гарячі клавіші:

* Масштабування: Затисніть Ctrl і крутіть коліщатко миші (Scroll) для збільшення або зменшення схеми без втрати якості.

* Панель інструментів (Toolbar): У верхній частині вікна розташовані кнопки для швидкого оновлення діаграми (Refresh), перемикання між векторним (SVG) та растровим (PNG) форматами, копіювання в буфер обміну та збереження картинки на диск.
